## 1. 프로젝트 개요

### 1.1 프로젝트 목적

케어매치(CareMatch) V3는 요양병원 및 요양원에서 근무하는 간병인과 환자 보호자를 연결하는 **구인구직 매칭 플랫폼**입니다. 본 시스템은 다음의 목적을 가지고 개발되었습니다:

- 검증된 간병인과 보호자 간의 신뢰할 수 있는 매칭 서비스 제공
- 50-70세 고령 사용자를 위한 접근성 최적화 UI/UX 구현
- 실시간 채팅을 통한 원활한 커뮤니케이션 지원
- 투명한 리뷰 시스템을 통한 서비스 품질 보장

### 1.2 프로젝트 범위

| 구분 | 상세 내용 |
|------|----------|
| 대상 사용자 | 간병인, 보호자 (주 연령층: 50-70세) |
| 서비스 지역 | 대한민국 전역 |
| 플랫폼 | 웹 (반응형, 모바일 최적화) |
| 개발 기간 | 20일 (5단계 개발) |

### 1.3 프로젝트 일정

| 단계 | 기간 | 주요 작업 | 완료 상태 |
|------|------|----------|----------|
| Phase 1 | Day 1-2 | 프로젝트 셋업, 환경 구성 | 완료 |
| Phase 2 | Day 3-5 | 인증 시스템 (소셜 로그인) | 완료 |
| Phase 3 | Day 6-12 | 핵심 기능 (대시보드, 구인/지원, 리뷰) | 완료 |
| Phase 4 | Day 13-16 | 채팅 시스템 (실시간 메시징) | 완료 |
| Phase 5 | Day 17-20 | 배포 및 최적화 | 완료 |

## 3. 주요 기능

### 3.1 사용자 유형별 기능

#### 간병인 (Caregiver)

| 기능 | 설명 |
|------|------|
| 일자리 검색 | 지역, 급여, 근무형태별 구인글 검색 |
| 일자리 지원 | 관심 있는 구인글에 지원서 제출 |
| 지원 현황 관리 | 제출한 지원서 상태 확인 및 관리 |
| 프로필 관리 | 경력, 자격증, 자기소개 등록/수정 |
| 실시간 채팅 | 보호자와 1:1 채팅 상담 |
| 리뷰 확인 | 받은 리뷰 및 평점 확인 |

#### 보호자 (Guardian)

| 기능 | 설명 |
|------|------|
| 구인글 등록 | 간병 조건, 급여, 근무지 등 구인 정보 등록 |
| 구인글 관리 | 등록한 구인글 수정, 마감, 삭제 |
| 지원자 관리 | 지원자 목록 확인 및 채용/거절 처리 |
| 간병인 검색 | 조건에 맞는 간병인 프로필 검색 |
| 실시간 채팅 | 간병인과 1:1 채팅 상담 |
| 리뷰 작성 | 채용한 간병인에 대한 리뷰 작성 |

### 3.2 공통 기능

| 기능 | 설명 |
|------|------|
| 소셜 로그인 | 카카오, 네이버 계정으로 간편 로그인 |
| 프로필 완성 | 회원가입 후 상세 프로필 등록 |
| 대시보드 | 사용자 유형별 맞춤 대시보드 |
| 알림톡 | 주요 이벤트 발생 시 카카오 알림톡 발송 |

## 4. 접근성 특징

### 4.1 고령 사용자 최적화

본 시스템은 50-70세 고령 사용자를 주 대상으로 하여 다음과 같은 접근성 최적화를 적용하였습니다:

| 항목 | 적용 내용 |
|------|----------|
| 폰트 크기 | 최소 14px, 기본 16px 이상 |
| 버튼 크기 | 최소 터치 영역 40x40px 이상 |
| 색상 대비 | WCAG AA 기준 충족 (4.5:1 이상) |
| 여백 | 충분한 요소 간 간격 |
| 아이콘 | 텍스트와 함께 사용 |
| 에러 메시지 | 친절하고 명확한 안내 |

### 4.2 반응형 디자인

- 모바일 (320px ~ 767px)
- 태블릿 (768px ~ 1023px)
- 데스크톱 (1024px 이상)
