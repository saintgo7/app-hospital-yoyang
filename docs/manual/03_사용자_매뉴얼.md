## 1. 서비스 소개

### 1.1 케어매치란?

케어매치는 요양병원, 요양원에서 일하실 **간병인**과 간병 서비스가 필요한 **보호자**를 연결해드리는 서비스입니다.

### 1.2 주요 특징

| 특징 | 설명 |
|------|------|
| **검증된 간병인** | 자격증과 경력이 검증된 전문 간병인만 활동합니다 |
| **실시간 채팅** | 간병인과 보호자가 직접 대화하며 상담할 수 있습니다 |
| **솔직한 리뷰** | 실제 이용자의 후기로 신뢰할 수 있습니다 |
| **간편한 사용** | 큰 글씨와 쉬운 메뉴로 누구나 편하게 사용할 수 있습니다 |

## 2. 회원가입 및 로그인

### 2.1 회원가입 방법

케어매치는 카카오 또는 네이버 계정으로 간편하게 가입할 수 있습니다.

#### 단계별 안내

**1단계: 홈페이지 접속**
- 웹브라우저에서 케어매치 홈페이지에 접속합니다
- 화면 오른쪽 상단의 **[회원가입]** 버튼을 누릅니다

**2단계: 회원 유형 선택**
- **간병인으로 시작하기**: 일자리를 찾으시는 분
- **보호자로 시작하기**: 간병인을 찾으시는 분

**3단계: 소셜 로그인**
- **카카오로 시작하기** 또는 **네이버로 시작하기** 버튼을 누릅니다
- 해당 서비스에서 로그인합니다

**4단계: 프로필 완성**
- 이름, 연락처 등 기본 정보를 입력합니다
- **[완료]** 버튼을 눌러 가입을 완료합니다

### 2.2 로그인 방법

**1단계**: 화면 오른쪽 상단의 **[로그인]** 버튼을 누릅니다

**2단계**: 가입 시 사용한 소셜 계정(카카오/네이버)으로 로그인합니다

**3단계**: 로그인 완료 후 대시보드로 이동합니다

### 2.3 로그아웃 방법

화면 오른쪽 상단의 프로필 옆 **[로그아웃]** 버튼을 누르시면 됩니다.

## 3. 간병인 사용 가이드

### 3.1 대시보드

로그인 후 보이는 첫 화면입니다. 여기서 다음 정보를 확인할 수 있습니다:

| 항목 | 설명 |
|------|------|
| 지원 현황 | 내가 지원한 일자리 수와 상태 |
| 새 일자리 | 최근 등록된 구인 정보 |
| 받은 리뷰 | 보호자가 남긴 평가 |

### 3.2 일자리 찾기

**1단계**: 상단 메뉴에서 **[구인 정보]**를 누릅니다

**2단계**: 원하는 조건으로 일자리를 검색합니다
- 지역 선택
- 급여 조건
- 근무 형태 (입주/출퇴근/시간제)

**3단계**: 마음에 드는 구인글을 눌러 자세한 내용을 확인합니다

### 3.3 일자리 지원하기

**1단계**: 구인글 상세 페이지에서 **[지원하기]** 버튼을 누릅니다

**2단계**: 지원 메시지를 작성합니다
- 자신을 소개하는 내용
- 해당 일자리에 관심을 가진 이유
- 경력이나 특기 사항

**3단계**: **[지원 완료]** 버튼을 누릅니다

### 3.4 지원 현황 확인

**1단계**: 대시보드에서 **[지원 현황]**을 누릅니다

**2단계**: 지원 상태를 확인합니다

| 상태 | 의미 |
|------|------|
| 검토중 | 보호자가 지원서를 확인하는 중 |
| 수락됨 | 보호자가 지원을 수락함 (채팅 가능) |
| 거절됨 | 아쉽지만 이번에는 선택되지 않음 |

### 3.5 프로필 관리

좋은 일자리를 찾으려면 프로필을 잘 작성하는 것이 중요합니다.

**프로필 수정 방법**:
1. 대시보드에서 **[내 프로필]**을 누릅니다
2. 수정할 내용을 변경합니다
3. **[저장]** 버튼을 누릅니다

**작성 팁**:
- 경력 사항을 자세히 적어주세요
- 보유 자격증을 모두 등록해주세요
- 자기소개는 진심을 담아 작성해주세요
- 프로필 사진은 밝고 친근한 인상으로 해주세요

## 4. 보호자 사용 가이드

### 4.1 대시보드

로그인 후 보이는 첫 화면입니다. 여기서 다음 정보를 확인할 수 있습니다:

| 항목 | 설명 |
|------|------|
| 내 구인글 | 등록한 구인글과 지원자 수 |
| 새 지원자 | 최근 지원한 간병인 목록 |
| 진행중인 채팅 | 간병인과의 대화 목록 |

### 4.2 구인글 등록하기

**1단계**: 대시보드에서 **[구인글 작성]** 버튼을 누릅니다

**2단계**: 구인 정보를 입력합니다

| 항목 | 입력 내용 |
|------|----------|
| 제목 | 한눈에 알아볼 수 있는 제목 |
| 근무지 | 실제 근무할 장소 (요양병원/요양원 이름) |
| 급여 | 월급 또는 일당 |
| 근무 형태 | 입주 / 출퇴근 / 시간제 |
| 환자 정보 | 환자 상태와 필요한 케어 내용 |
| 상세 설명 | 기타 근무 조건이나 요청사항 |

**3단계**: **[등록하기]** 버튼을 누릅니다

### 4.3 지원자 관리

**1단계**: 대시보드에서 **[내 구인글]**을 누릅니다

**2단계**: 구인글을 선택하면 지원자 목록이 보입니다

**3단계**: 지원자 프로필을 확인합니다
- 경력 사항
- 자격증
- 자기소개
- 받은 리뷰

**4단계**: 마음에 드는 지원자에게 **[채팅하기]** 또는 **[수락]**을 누릅니다

### 4.4 구인글 수정/마감

**구인글 수정**:
1. 구인글 상세 페이지에서 **[수정]** 버튼을 누릅니다
2. 내용을 수정합니다
3. **[저장]** 버튼을 누릅니다

**구인글 마감**:
- 간병인을 구했다면 **[마감]** 버튼을 눌러 더 이상 지원을 받지 않을 수 있습니다

## 5. 채팅 기능

### 5.1 채팅 시작하기

채팅은 다음 경우에 가능합니다:
- 보호자가 지원자를 수락한 경우
- 구인글 상세에서 **[채팅하기]** 버튼을 누른 경우

### 5.2 채팅 사용법

**1단계**: 상단 또는 대시보드에서 **[채팅]**을 누릅니다

**2단계**: 대화할 상대를 선택합니다

**3단계**: 메시지를 입력하고 **[보내기]**를 누릅니다

### 5.3 채팅 주의사항

- 상대방을 존중하는 대화를 해주세요
- 개인정보(주민번호, 계좌번호 등)는 공유하지 마세요
- 문제가 있으면 고객센터에 신고해주세요

## 6. 리뷰 시스템

### 6.1 리뷰 작성하기 (보호자)

간병 서비스가 완료되면 간병인에 대한 리뷰를 남길 수 있습니다.

**1단계**: 대시보드에서 **[리뷰 작성]**을 누릅니다

**2단계**: 별점을 선택합니다 (1~5점)

**3단계**: 솔직한 후기를 작성합니다

**4단계**: **[등록]** 버튼을 누릅니다

### 6.2 리뷰 확인하기 (간병인)

**1단계**: 대시보드에서 **[받은 리뷰]**를 누릅니다

**2단계**: 보호자가 남긴 리뷰와 평점을 확인합니다

### 6.3 리뷰 작성 가이드

좋은 리뷰 예시:
- "어머니를 정성껏 돌봐주셔서 감사합니다. 항상 밝은 모습으로 대해주셔서 어머니도 좋아하셨어요."
- "시간 약속을 잘 지키시고, 환자 상태를 자세히 알려주셔서 안심이 됐습니다."

## 7. 자주 묻는 질문 (FAQ)

### Q1. 가입비나 이용료가 있나요?
**A**: 기본 서비스는 무료입니다. 일부 프리미엄 기능은 유료일 수 있습니다.

### Q2. 개인정보는 안전한가요?
**A**: 네, 모든 개인정보는 암호화되어 안전하게 보관됩니다.

### Q3. 연락처가 상대방에게 바로 공개되나요?
**A**: 아니요, 채팅을 통해 먼저 대화한 후 서로 동의하면 연락처를 교환할 수 있습니다.

### Q4. 문제가 있는 사용자를 신고할 수 있나요?
**A**: 네, 고객센터를 통해 신고하시면 확인 후 조치하겠습니다.

### Q5. 비밀번호를 잊어버렸어요.
**A**: 케어매치는 소셜 로그인(카카오/네이버)을 사용하므로, 해당 서비스에서 비밀번호를 재설정해주세요.
