# PRD (Product Requirements Document)
# CareMatch V3 - 제품 요구사항 정의서

---

## 1. 제품 개요

### 1.1 제품 비전
> "간병인과 보호자를 신뢰할 수 있는 플랫폼에서 연결하여, 양질의 간병 서비스 접근성을 높인다"

### 1.2 목표 시장
- **1차 타겟**: 수도권 요양병원/요양원 및 개인 가정
- **2차 타겟**: 전국 확대

### 1.3 핵심 문제 정의

| 이해관계자 | 현재 문제점 |
|-----------|------------|
| **간병인** | 구인 정보 분산, 중개 수수료 부담, 검증 어려움 |
| **보호자** | 신뢰할 수 있는 간병인 찾기 어려움, 직접 계약 부담 |
| **시설** | 인력 수급 불안정, 채용 프로세스 비효율 |

## 2. 사용자 페르소나

### 2.1 간병인 - 김순자 (58세, 여성)
- **배경**: 10년 경력 간병인, 요양보호사 자격증 보유
- **목표**: 안정적인 일자리, 적정 급여, 편한 근무 환경
- **불편함**: 스마트폰 사용 미숙, 복잡한 앱 조작 어려움
- **니즈**: 큰 글씨, 간단한 UI, 전화 연결 기능

### 2.2 개인 보호자 - 이정희 (52세, 여성)
- **배경**: 80대 어머니 간병인 필요
- **목표**: 믿을 수 있는 간병인, 합리적 비용
- **불편함**: 간병인 신원/경력 확인 어려움
- **니즈**: 인증된 간병인, 후기 확인, 빠른 연결

### 2.3 시설 관리자 - 박관리 (45세, 남성)
- **배경**: 50병상 요양원 행정팀장
- **목표**: 안정적 인력 수급, 채용 업무 효율화
- **불편함**: 개별 연락, 이력서 관리 번거로움
- **니즈**: 일괄 공고, 지원자 관리, 이력 조회

### 3.2 상세 기능 명세

#### F001: 소셜 로그인
```
설명: 카카오/네이버 계정으로 간편 로그인
흐름:
  1. 로그인 버튼 클릭
  2. OAuth 제공자 선택 (카카오/네이버)
  3. 제공자 인증 페이지로 리다이렉트
  4. 인증 완료 후 콜백
  5. 신규 사용자 → 회원가입 페이지
  6. 기존 사용자 → 대시보드
데이터:
  - email: 이메일 (필수)
  - name: 이름
  - profile_image: 프로필 이미지
  - auth_provider: kakao | naver
```

#### F003: 간병인 프로필
```
설명: 간병인의 상세 프로필 정보 관리
필드:
  - 기본정보: 성별, 생년월일, 연락처
  - 주소: 시/도, 시/군/구
  - 경력: 총 경력 년수, 경력 상세
  - 자격증: 보유 자격증 목록
  - 희망조건:
    - 근무형태: 입주, 출퇴근, 시간제 (복수선택)
    - 희망지역: 시/도, 시/군/구 (복수선택, 최대 5개)
    - 희망급여: 유형(시급/일급/월급), 최소~최대
    - 근무가능일: 날짜
  - 자기소개: 500자 이내
기능:
  - 프로필 완성도 표시
  - 프로필 공개/비공개 설정
```

#### F005: 구인공고 CRUD
```
설명: 보호자가 간병인 구인공고 관리
필드:
  - 제목: 50자 이내
  - 근무형태: 입주 | 출퇴근 | 시간제
  - 근무지역: 시/도, 시/군/구, 상세주소
  - 근무기간: 시작일, 종료일(선택), 기간유형
  - 근무시간: 시작시간, 종료시간 (출퇴근/시간제)
  - 급여: 유형, 금액, 협의가능 여부
  - 환자정보: 성별, 연령대, 진단명, 거동/인지 상태
  - 우대조건: 희망성별, 연령대, 필수자격증
  - 제공사항: 식사제공, 숙소제공
  - 상세설명: 1000자 이내
  - 긴급여부: 급구 표시
상태:
  - active: 모집중
  - closed: 마감
  - matched: 매칭완료
  - deleted: 삭제됨
```

#### F009: 실시간 채팅
```
설명: 간병인-보호자 간 1:1 실시간 채팅
기능:
  - 텍스트 메시지 전송
  - 이미지 전송 (최대 5MB)
  - 읽음 표시
  - 새 메시지 알림
  - 채팅방 목록 (최근 메시지순)
  - 안읽은 메시지 수 표시
기술:
  - Supabase Realtime 활용
  - 실시간 구독 (Subscribe)
```

## 4. 사용자 시나리오

### 4.1 간병인 시나리오

#### 시나리오 A: 첫 가입 및 프로필 작성
```
1. 앱 접속 → 로그인 페이지
2. "카카오로 시작하기" 클릭
3. 카카오 로그인 완료
4. 회원 유형 선택: "간병인으로 가입"
5. 기본정보 입력 (이름, 연락처, 성별, 생년월일)
6. 경력 정보 입력 (경력년수, 상세경력)
7. 자격증 등록 (이미지 업로드)
8. 희망조건 설정 (지역, 급여, 근무형태)
9. 자기소개 작성
10. 프로필 완성 → 대시보드 이동
```

#### 시나리오 B: 일자리 검색 및 지원
```
1. 대시보드 → "일자리 찾기"
2. 필터 설정 (서울 강남구, 입주, 월 300만원 이상)
3. 검색 결과 목록 확인
4. 관심 공고 클릭 → 상세 페이지
5. 환자정보, 근무조건 확인
6. "지원하기" 클릭
7. 자기소개서 작성 (선택)
8. 지원 완료 → 알림톡 수신 (보호자에게)
9. 채팅 요청 대기
```

### 4.2 보호자 시나리오

#### 시나리오 C: 구인공고 등록
```
1. 로그인 → 보호자 대시보드
2. "공고 등록하기" 클릭
3. 환자 정보 입력/선택
4. 근무 조건 입력 (지역, 기간, 시간, 급여)
5. 우대 조건 설정
6. 상세 설명 작성
7. 미리보기 확인
8. "공고 등록" 클릭
9. 공고 게시 완료
```

#### 시나리오 D: 지원자 검토 및 채팅
```
1. 대시보드 → 내 공고 목록
2. 공고 선택 → 지원자 N명 확인
3. 지원자 목록에서 프로필 클릭
4. 간병인 상세 프로필 확인 (경력, 자격증, 리뷰)
5. "채팅하기" 클릭
6. 채팅방 생성 → 대화 시작
7. 상세 조건 협의
8. 적합 시 "채용 확정" 클릭
9. 매칭 완료 → 알림톡 발송
```

## 6. 수익 모델

### 6.1 무료 기본 서비스
- 회원가입, 프로필 작성
- 구인공고 등록 (월 3건)
- 기본 검색 및 지원
- 채팅 기능

### 6.2 프리미엄 구독 (예정)

| 플랜 | 대상 | 월 요금 | 혜택 |
|------|------|---------|------|
| 간병인 프리미엄 | 간병인 | ₩4,900 | 프로필 상위 노출, 지원 우선 알림 |
| 보호자 프리미엄 | 보호자 | ₩9,900 | 무제한 공고, 간병인 직접 연락 |
| 시설 프리미엄 | 요양시설 | ₩29,900 | 무제한 공고, 다중 관리자, 통계 |

### 6.3 중개 수수료 (선택)
- 매칭 성사 시 첫 달 급여의 5-10%
- 무료 직접 계약 옵션 제공
