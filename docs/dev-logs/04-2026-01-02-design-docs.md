## 완료된 작업

### 1. shadcn/ui 표준 디자인 변경

**문제점**: 커스텀 Tailwind 클래스(`text-accessible-*`, `min-h-touch-*`)가 shadcn/ui 표준과 맞지 않아 유지보수 어려움

**해결**:
- `globals.css`: CSS 변수 기반 색상 시스템으로 변경
- `tailwind.config.ts`: 커스텀 클래스 제거, 표준 shadcn/ui 설정 적용
- UI 컴포넌트 7개 표준화:
  - Button, Card, Input, Label, Textarea, Badge, Avatar
- 페이지 클래스 변경:
  - `text-accessible-*` → `text-sm/base/lg/xl`
  - `min-h-touch-*` → `h-10/h-12`
  - `size="xl"` → `size="lg"`

**커밋**: `8ba2895` - refactor: shadcn/ui 표준 디자인으로 변경
